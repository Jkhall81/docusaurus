# Host Operating System

This application runs on Linux. The fog server installed here is hosted on a machine running Ubuntu Server 24.04.1.

## Distros supported

- Ubuntu
- Debian
- CentOS
- Red Hat
- Fedora
- Arch

## Other Requirements

- You need to have access to your DHCP server...... or
- Use Dnsmasq
- have git installed
